# Support

## Where To Go

| Need | Route |
|---|---|
| Bug report or code issue | GitHub Issues on the private repo |
| Evidence/path contradiction | GitHub Issues with file path and evidence |
| Security vulnerability | Private report to `architects@zer0pa.ai` |
| Licensing question | See `../LICENSE`; escalate unresolved questions to `architects@zer0pa.ai` |
| IMC compatibility question | Read `family/BIO_IMC_ALIGNMENT_REPORT.md` first |

## Response Expectations

- Evidence-backed bug reports get priority.
- Security reports should stay private.
- Questions already answered in `README.md`, `FAQ.md`, or `PROOF_INDEX.md` may be redirected there.
